# Sorting-Visualizer
Made Sorting Algorithm visualizer in html,css and js.
Following algorithms are currently supported:
1. Merge sort
2. Quick sort(last pivot)
3. Quick sort(dual pivot)
4. Heap sort
5. Bubble sort
6. Selection sort
7. Insertion sort

Goto: https://jenil123.github.io/Sorting-Visualizer/
